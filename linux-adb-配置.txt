一 安裝adb和fastboot
sudo apt-get update
sudo apt-get install android-tools-adb android-tools-fastboot

二 配置USB裝置
1 查看USB裝置訊息
$ lsusb
Bus 001 Device 007: ID 12d1:1038 Huawei Technologies Co., Ltd. Ideos (debug mode)

2 編輯 rules

sudo leafpad /etc/udev/rules.d/51-android.rules
添加如下内容：
# adb and fastboot protocol
SUBSYSTEM=="usb", ATTR{idVendor}=="12d1", MODE="0666", GROUP="plugdev"

sudo chmod a+x /etc/udev/rules.d/51-android.rules

3.更新系統
sudo udevadm control --reload-rules
或
sudo service udev restart

sudo chown root:root /usr/bin/adb
sudo chmod a+x /usr/bin/adb
sudo chmod a+s /usr/bin/adb
adb kill-server
adb server
